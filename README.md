👾 PixelNav Station (像素导航站)

一个致敬 Windows 98 风格、单文件、无后端、支持自导出的复古 Web3 导航站。

PixelNav Station 是一个纯前端的导航页解决方案。它将所有数据、样式和逻辑封装在一个单独的 index.html 文件中。您不需要数据库，不需要服务器后台，甚至不需要懂得编程，即可拥有一个极具个性的个人导航站。

✨ 核心特性

💾 单文件架构 (Serverless)
无需数据库，无需后端语言（PHP/Python/Node.js）。
数据直接“刻录”在 HTML 文件内部，即插即用。
🎨 Windows 98 像素风
经典的灰色窗口、蓝天渐变标题栏、复古滚动条和像素字体。
完美的移动端与桌面端自适应布局。
🔐 内置管理后台
前端加密登录（SHA-256），保护管理权限。
支持可视化添加、修改、删除分类和链接。
批量导入：支持简单的文本格式批量添加链接。
⚙️ 高度自定义
自定义窗口标题、搜索框提示语。
自定义菜单栏（支持超链接跳转）。
系统运行时间：自定义起始时间，自动计算“系统已运行”时长。
🚀 独特的“导出部署”模式
在网页上修改完内容后，点击“导出”，系统会自动生成一个包含最新数据的 index.html。
将新文件上传到 GitHub Pages、Vercel 或任何静态服务器即可完成发布。
🥚 Web3 彩蛋
尝试点击“开始”菜单里的“关闭系统”，会有惊喜。
🚀 快速开始

下载：获取 index.html 文件。
打开：双击在浏览器中直接打开。
登录：
点击左下角 Start 按钮。
点击 管理员登录。
输入默认密码：admin。
编辑：
登录后，您可以点击工具栏的图标来添加分类或链接。
鼠标悬停在现有项目上，会出现 编辑(✎) 和 删除(×) 按钮。
保存：
重要：修改完成后，点击工具栏最左侧的 "导出" (Save) 按钮。
浏览器会下载一个新的 index.html。
用这个新文件覆盖您服务器上的旧文件，即完成更新。
🛠️ 管理员指南

1. 全局设置 (Settings)

登录后点击工具栏的 ⚙️ 图标，可以修改：

窗口标题：浏览器的标签页标题和页面内窗口标题。
菜单栏：格式为 名称|链接，多个菜单用逗号分隔。
例如：首页|/, 关于|https://github.com, 摸鱼
起始时间：设置站点的建立时间，左下角会自动显示“系统已运行 X天X时...”。
2. 批量导入 (Batch Import)

点击工具栏的 ➕📝 图标，打开批量导入窗口。 格式非常简单，一行一个：

Google | google.com | 搜索引擎
Bilibili | bilibili.com | 视频网站
系统会自动补全 https:// 并去重。

3. 图标设置

在添加/修改链接时，图标 (Icon) 栏支持两种格式：

Emoji：直接输入 🔥、🚀、💻 等表情符号。
图片 URL：输入以 http 开头的图片链接，会自动渲染为图片。
🔒 安全说明

密码存储：管理员密码经过 SHA-256 哈希处理存储在代码中，查看源代码无法直接看到明文密码。
隐私保护：登录弹窗移除了“默认密码是admin”的提示，防止陌生人尝试。
注意：由于是纯前端验证，此安全机制防君子不防黑客。请勿在站点内存储高度机密的信息。
📂 数据结构

所有数据存储在 HTML 文件头部的 <script id="nav-data" type="application/json"> 标签内。 如果您是开发者，可以直接编辑这部分 JSON：

{
    "config": {
        "windowTitle": "我的导航站",
        "menus": ["首页|/", "GitHub|[https://github.com](https://github.com)"],
        "statusPrefix": "安全运行",
        "startTime": "2023-01-01T00:00:00.000Z"
    },
    "categories": [
        {
            "id": "cat-1",
            "name": "常用工具",
            "links": [...]
        }
    ]
}

无任何构建过程，所见即所得。
📝 修改默认密码

如果您想修改默认密码 admin，请按以下步骤操作：

使用 SHA-256 在线工具 生成您新密码的哈希值。
用文本编辑器打开 index.html。
搜索 const ADMIN_HASH_SHA256。
将引号内的字符串替换为您生成的新哈希值。
保存文件。
