👾 PixelNav Station

极简、复古、Web3 风格的单文件导航站生成器。
无需数据库，无需服务器后台，一个 HTML 文件即是一切。

🚨 V29.0 版本更新日志 (Critical Fixes)
此版本是针对 交互冲突 和 导入逻辑 的最终修正版，强烈建议升级。
✅ 核心修复
• 📥 导入功能终极修复
• 强制初始化：修复了打开导入窗口时下拉菜单为空的 BUG。现在系统会自动扫描并选中第一个可用的文件夹。
• 空值保护：如果您的站点没有任何二级分类，点击导入按钮时会直接拦截并提示“请先新建分类”，防止程序报错卡死。
• 🛡️ 弹窗层级管理 (Z-Index)
• 彻底解决弹窗乱跳：重新设计了 CSS 层级。
• 错误提示 (Z=200) > 导入窗口 (Z=150) > 普通窗口 (Z=100)。
• 确保在任何情况下，错误提示永远在最上层，不会被其他窗口遮挡。
• 🖱️ 点击穿透修复
• 给工具栏所有按钮添加了强制的 stopPropagation，防止点击按钮时误触发底部的网页跳转。
✨ 核心特性
1. 💾 单文件架构 (Serverless)
• 无需数据库，无需后端语言。
• 数据直接“刻录”在 HTML 文件内部，即插即用。
• 部署只需一步：把 HTML 文件上传到 GitHub Pages、Vercel 或任何服务器即可。
2. 🎨 纯正 Windows 98 风
• 经典的灰阶界面、蓝天渐变标题栏、复古滚动条。
• 左侧侧边栏文字完美垂直居中。
3. 🔃 稳定的排序系统
• 点击 排序 (Sort) 按钮进入专用模式。
• 使用 ⬆⬇ 箭头 进行移动，比拖拽更精准，绝无误触。
4. 📝 强大的批量导入
• 支持 名称 | 网址 | 描述 格式。
• 支持直接从 Excel 复制粘贴（自动识别制表符）。
• 支持粘贴纯网址（自动补全标题）。
5. 🔐 安全与隐私
• 管理员密码采用 SHA-256 哈希存储，源码不含明文。
• 登录界面移除“默认密码”提示。
📖 快速上手指南
1. 启动与登录
1. 下载 index.html 文件。
2. 双击直接在浏览器打开。
3. 点击左下角 Start -> 管理员登录。
4. 默认密码：admin
2. 构建您的导航站
• 新建组/文件夹：点击工具栏的 📂+ 图标。
• 提示：必须先建一个“一级大类”，再建“二级文件夹”，链接只能存放在二级文件夹中。
• 添加链接：选中一个二级文件夹后，点击 🔗+ 图标。
• 批量导入：点击 📝+ 图标，粘贴您的网址列表。
• 注意：如果导入没反应，请检查是否已创建并选中了二级文件夹。
3. 保存与发布 (最重要的一步！)
• 修改完成后，务必点击工具栏最左侧的 "💾 导出" 按钮。
• 浏览器会下载一个新的 index.html。
• 用这个新文件覆盖您服务器上的旧文件，即完成更新。
