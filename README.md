## 🚨 V28.0 版本更新日志 (Critical Fixes)

此版本重点修复了 **批量导入** 和 **移动端交互** 的严重 Bug，建议所有旧版本用户立即升级。

### ✅ 核心修复

- **📥 导入功能重构**
    - **修复下拉菜单空白**：修复了打开导入窗口时，目标文件夹列表未正确生成的问题。
    - **智能兜底**：如果当前未选中任何文件夹，系统会自动定位到第一个可用分类，不再报错。
    - **空状态保护**：如果连一个二级分类都没有，点击导入会直接弹出提示“请先新建分类”，防止程序卡死。
- **🛡️ 弹窗层级冲突 (Z-Index)**
    - 修复了“导入窗口”与“错误提示”打架的问题。现在错误提示永远在最顶层，不会导致界面冻结。
- **📱 移动端/iOS 适配**
    - 优化了下拉选择框在 iPhone 上的显示样式。
    - 增加了弹窗的安全边距，防止被软键盘遮挡。

## ✨ 核心特性

1. **💾 单文件架构 (Serverless)**
    - 无需数据库，无需后端语言（PHP/Python/Node.js）。
    - 数据直接“刻录”在 HTML 文件内部，即插即用。
    - **部署只需一步**：把 HTML 文件上传到 GitHub Pages、Vercel 或任何服务器即可。
2. **🎨 纯正 Windows 98 风**
    - 经典的灰阶界面、蓝天渐变标题栏、复古滚动条。
    - 左侧侧边栏文字完美垂直居中。
3. **🔃 稳定的排序系统**
    - 点击 **排序 (Sort)** 按钮进入专用模式。
    - 使用 **⬆⬇ 箭头** 进行移动，比拖拽更精准，绝无误触。
4. **📝 强大的批量导入**
    - 支持 `名称 | 网址 | 描述` 格式。
    - 支持直接从 **Excel** 复制粘贴（自动识别制表符）。
    - 支持粘贴纯网址（自动补全标题）。
5. **🔐 安全与隐私**
    - 管理员密码采用 **SHA-256** 哈希存储，源码不含明文。
    - 登录界面移除“默认密码”提示。

## 📖 快速上手指南

### 1. 启动与登录

1. 下载 `index.html` 文件。
2. 双击直接在浏览器打开。
3. 点击左下角 **Start** → **管理员登录**。
4. 默认密码：`admin`

### 2. 构建您的导航站

- **新建组/文件夹**：点击工具栏的 `📂+` 图标。  
  *建议先建一个“一级大类”，再建“二级文件夹”。*
- **添加链接**：选中一个文件夹，点击 `🔗+` 图标。
- **批量导入**：点击 `📝+` 图标，粘贴您的网址列表。

### 3. 保存与发布 (重要！)

- 修改完成后，**务必点击工具栏最左侧的 "💾 导出" 按钮**。
- 浏览器会下载一个新的 `index.html`。
- **用这个新文件覆盖您服务器上的旧文件**，即完成更新。

## 🔒 如何修改密码？

默认密码是 `admin`。如需修改：

1. 使用 [SHA-256 在线工具](https://www.google.com/search?q=sha256+online) 生成新密码的哈希值。
2. 用记事本或代码编辑器打开 `index.html`。
3. 搜索关键词：`const ADMIN_HASH_SHA256`。
4. 将引号内的值替换为您生成的新哈希值。
5. 保存文件。

## ❓ 常见问题 (FAQ)

**Q: 为什么点击导入按钮没反应？**  
A: 请确保使用的是 **V28.0** 及以上版本。旧版本可能因为未选中文件夹而报错，新版本已自动修复。

**Q: 为什么在手机上有些按钮点不动？**  
A: V28.0 修复了移动端层级问题。如仍有问题，请刷新页面或清除浏览器缓存。

**Q: 部署后数据会丢吗？**  
A: 不会。只要每次修改后都点击了 **"导出"** 并上传了新文件，数据就永久保存在 HTML 代码里了。
